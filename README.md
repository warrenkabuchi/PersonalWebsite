# Warren Kabuchi - Personal Website

This is the personal portfolio website for Warren Kabuchi, AI Engineer.

## Project Structure

- `index.html`: Main entry point containing all site content and styles.
- `assets/`: (Optional) Directory for images and other static assets.

## Local Development

To view the site locally, simply open `index.html` in your web browser.

## Deployment

This site is ready to be deployed to any static site hosting provider (Vercel, Netlify, GitHub Pages, etc.).

### Vercel

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory.

## Note on API Keys

This project currently uses a client-side API key for the Gemini demo. For production use, ensure you have restricted this key in your Google Cloud Console to only allow requests from your deployed domain.
